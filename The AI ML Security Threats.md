### Machine Learning (ML) 
### It enables systems to learn from data and improve accuracy over time.
### It follows an iterative lifecycle from problem definition to deployment and continuous improvement.

### ML is a subfield of AI that learns without explicit instructions
### Performance improves with more data and experience
### Lifecycle includes problem definition, data preparation, training, and evaluation
### Models are deployed for real-world use and continuously monitored

### ML algorithms 
### It create models using decision processes, error functions, and optimization
### Iterative learning improves accuracy until desired performance is reached
### Supervised : learning uses labeled data for prediction tasks
### Unsupervised : learning finds patterns in unlabeled data
### Semi-supervised : learning combines labeled and unlabeled data
### Reinforcement : learning improves through rewards and penalties

### Neural networks 
### It mimic the human brain by using interconnected nodes to process information.
### Deep learning builds on neural networks to enable advanced pattern recognition and learning.
### Neural networks are inspired by how the human brain functions
### They use interconnected nodes similar to neurons and synapses
<img width="1140" height="400" alt="image" src="https://github.com/user-attachments/assets/6e8b43b2-4452-41ce-8be6-48b969ac3e9b" />

### Hidden layers process and refine the information
### Connections act like synapses with assigned weights
### Weights define the importance of each input feature
### Output layer generates the final prediction
<img width="800" height="800" alt="image" src="https://github.com/user-attachments/assets/852fec4d-5a0a-4e2b-91af-16b3340a2eb6" />

### Deeper layers combine features to form complete patterns
### Output layer selects the most likely prediction (e.g., digit classification)
### Each digit has a corresponding output node for probability scoring
### Networks with more than three layers are called Deep Learning models

### Deep Learning is a scalable form of ML that learns from raw, unlabelled data using neural networks.
### Its growth is driven by large datasets and removes the need for human-labelled data.
### Both ML and DL take input data and produce predictions or classifications
### ML typically relies on labelled data, while DL can work with unlabelled data
### DL automatically extracts features from raw, unstructured data
### It reduces the need for human intervention in data preparation

### ATLAS maps threats specific to AI and machine learning systems

### These vulnerabilities can be exploited by attackers to change outputs, steal intelligence, or reduce model accuracy and reliability.
